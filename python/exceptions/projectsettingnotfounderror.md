# ProjectSettingNotFoundError

- 扩展: [BaseError](./baseerror.md)

**ProjectSettingNotFoundError is raised when project setting file 'swifthorse.yaml' is missing.**

- [message](#message)
- [stacktrace](#stacktrace)


### message
- 类型: str

错误消息。


### stacktrace
- 类型: str

call stack详细信息，基于此可以定位到失败的函数位置。
